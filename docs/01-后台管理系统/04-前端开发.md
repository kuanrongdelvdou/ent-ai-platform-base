# 前端开发

## 技术栈

- Soybean Admin（Vue 3 + TypeScript + Vite）
- Naive UI 组件库
- Pinia 状态管理
- Vue Router（动态路由模式）
- UnoCSS
- 请求通过 Vite proxy 转发（`/proxy-default/` → `http://localhost:3000`）

---

## 路由模式

`VITE_AUTH_ROUTE_MODE=dynamic` — 路由由后端 `/route/getUserRoutes` 返回，前端动态注册。

菜单显示文本通过 i18n 解析：`route.<routeName>` → 中文/英文标签。

## 页面组件结构规范

每个管理页面拆分为三个文件（参照 `frontend-example/src/views/manage/`）：

```
views/system/<module>/
  index.vue                      # 列表页（表格 + 分页）
  modules/
    <module>-search.vue          # 搜索栏
    <module>-operate-drawer.vue  # 新增/编辑抽屉
```

# 数据库设计

## 数据库：PostgreSQL 15，ORM：Prisma 7

> Prisma 7 不支持在 schema 中配置 datasource URL，必须使用 driver adapter 初始化：
> ```typescript
> import { PrismaPg } from '@prisma/adapter-pg';
> const adapter = new PrismaPg({ connectionString: process.env.DATABASE_URL });
> ```

## 表结构（全部已实现）

### 部门表 `Department`
| 字段 | 类型 | 说明 |
|------|------|------|
| id | String (uuid) | 主键 |
| parentId | String? | 父部门 ID |
| name | String | 部门名称 |
| sort | Int | 排序 |
| leaderId | String? | 负责人 |
| status | Int | 1=启用 2=停用 |

### 用户表 `User`
| 字段 | 类型 | 说明 |
|------|------|------|
| id | String (uuid) | 主键 |
| username | String | 登录名，唯一 |
| password | String | bcrypt 加密 |
| realName | String? | 真实姓名 |
| email | String? | 邮箱 |
| phone | String? | 手机号 |
| deptId | String? | 所属部门 ID |
| status | Int | 1=启用 2=停用 |

### 角色表 `Role`
| 字段 | 类型 | 说明 |
|------|------|------|
| id | String (uuid) | 主键 |
| name | String | 角色名称 |
| code | String | 角色标识，唯一 |
| status | Int | 1=启用 2=停用 |
| remark | String? | 备注 |

### 用户角色关联表 `UserRole`
| 字段 | 类型 | 说明 |
|------|------|------|
| userId | String | 用户 ID（复合主键） |
| roleId | String | 角色 ID（复合主键） |

### 菜单表 `Menu`
| 字段 | 类型 | 说明 |
|------|------|------|
| id | String (uuid) | 主键 |
| parentId | String? | 父菜单 ID（null = 顶级） |
| name | String | 路由名称（对应 i18n key） |
| type | Int | 1=目录 2=页面 |
| path | String? | 路由路径 |
| component | String? | 组件路径 |
| icon | String? | 图标 |
| permission | String? | 按钮权限标识 |
| sort | Int | 排序 |
| status | Int | 1=启用 2=停用 |

### 角色菜单关联表 `RoleMenu`
| 字段 | 类型 | 说明 |
|------|------|------|
| roleId | String | 角色 ID（复合主键） |
| menuId | String | 菜单 ID（复合主键） |

### 字典类型表 `DictType`
| 字段 | 类型 | 说明 |
|------|------|------|
| id | String (uuid) | 主键 |
| name | String | 类型名称 |
| code | String | 类型标识，唯一 |
| status | Int | 1=启用 2=停用 |
| remark | String? | 备注 |

### 字典数据表 `DictItem`
| 字段 | 类型 | 说明 |
|------|------|------|
| id | String (uuid) | 主键 |
| dictTypeId | String | 所属字典类型 ID |
| label | String | 显示标签 |
| value | String | 数据值 |
| sort | Int | 排序 |
| status | Int | 1=启用 2=停用 |

### 系统参数表 `Config`
| 字段 | 类型 | 说明 |
|------|------|------|
| id | String (uuid) | 主键 |
| key | String | 参数键，唯一 |
| value | String | 参数值 |
| remark | String? | 备注 |

### 操作日志表 `OperationLog`
| 字段 | 类型 | 说明 |
|------|------|------|
| id | String (uuid) | 主键 |
| userId | String? | 操作人 ID |
| username | String? | 操作人名 |
| module | String? | 模块名称 |
| action | String? | 操作类型 |
| method | String? | HTTP 方法 |
| url | String? | 请求路径 |
| ip | String? | 客户端 IP |
| requestBody | Json? | 请求参数 |
| responseCode | Int? | 响应状态码 |
| duration | Int? | 耗时（ms） |
| createdAt | DateTime | 操作时间 |

### 登录日志表 `LoginLog`
| 字段 | 类型 | 说明 |
|------|------|------|
| id | String (uuid) | 主键 |
| userId | String? | 用户 ID |
| username | String? | 用户名 |
| ip | String? | 客户端 IP |
| userAgent | String? | 浏览器 UA |
| status | Int | 1=成功 2=失败 |
| message | String? | 消息 |
| createdAt | DateTime | 登录时间 |

## RBAC 关系图

```
用户 → UserRole → 角色 → RoleMenu → 菜单
用户 → Department（所属部门）
```

---

## 菜单 component 命名规则

- 目录：`layout.base`
- 页面：`layout.base$view.<route_name>`（如 `layout.base$view.system_user`）
- route_name 与 i18n key `route.<name>` 对应，用于菜单显示文本

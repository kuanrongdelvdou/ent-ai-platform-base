# 协作规划

## 目标

支持多人协同编辑公文，有版本管理、评论批注能力。

---

## 功能模块

### 1. 富文本编辑器
- 公文在线编写和编辑
- 支持标题、段落、表格、图片等格式
- 编辑器选型：Tiptap（推荐，可扩展，支持协作）

### 2. 实时协作编辑
- 多人同时编辑同一文档
- 实时显示其他用户的光标和选区
- 基于 **Yjs** CRDT 算法，冲突自动合并
- 传输层：WebSocket（NestJS + ws）

### 3. 版本管理
- 自动保存历史版本（每次保存/定时）
- 版本列表，可查看任意历史版本
- 版本对比（diff 高亮）
- 版本回退

### 4. 评论与批注
- 选中文字，添加批注
- 批注回复、解决
- 批注列表侧边栏

### 5. 文档权限
- 文档可见性（仅自己/指定人/部门/全员）
- 编辑权限（只读/可编辑/可管理）

---

## 技术方案

```
Tiptap（编辑器）+ Yjs（CRDT 协作）
  ↓ WebSocket
NestJS（y-websocket server 或自实现）
  ↓
PostgreSQL（文档内容持久化 + 版本快照）
```

---

## 说明

协作功能复杂度高，作为 **Phase 5** 独立交付，不阻塞前面各阶段。
